# 設計仕様: CLAUDE.md テンプレート全面再設計

**日付:** 2026-05-11
**対象ファイル:** `03_python-project/CLAUDE.md`

## 背景と目的

`03_python-project/CLAUDE.md` はPythonバックエンド業務システム向けのCLAUDE.mdテンプレートだが、以下の課題がある:

1. ガイド注釈（`>`ブロッククォート）が薄く、記入時に迷う箇所がある
2. 「ローカルセットアップ手順」と「環境変数一覧」セクションがない — 初期セッションでClaude が毎回聞くことになる
3. `プロジェクト固有の禁止事項` が例コメントのみで実記入欄がない

目的: Claude Code が新規プロジェクト開始時に即座にコンテキストを把握できる、高品質な空白テンプレートを作成する。

## 設計決定

### セクション構成

1. コマンド一覧（注釈改善）
2. プロジェクト概要（注釈改善 + NG/OK例追加）
3. ローカルセットアップ（新規追加）
4. 環境変数（新規追加）
5. 機能一覧（注釈改善 + NG/OK例追加）
6. アーキテクチャ（注釈改善）
7. 技術スタック（注釈改善）
8. 外部I/F仕様（注釈改善）
9. システムスケール要件（注釈改善）
10. Claudeの振る舞い（禁止事項に実記入欄追加）
11. 開発フェーズ（変更なし）
12. 参照ドキュメント索引（変更なし）

### 注釈改善方針

各セクションのガイド注釈にNG例・OK例を追加する。
