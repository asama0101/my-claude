_patch_config(session, body) -> bool / _cli_command(session, command) -> bool — client.py:193-249

平たく言うと: どちらも同型のリトライループ（最大 _MAX_RETRIES=3回、client.py:30）。HTTP 200 ならTrue、それ以外または aiohttp.ClientError なら logger.warning してリトライ、上限到達で False を返す（例外は送出しない。呼び出し元の apply_subport_config が bool を見て例外に変換する）。
